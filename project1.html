AniLite

Live Demo GitHub

AniLite is a full stack iMDb like website, but for anime.

  • Users can log in, view anime by genre or rating, and subscribe to the mailing list of an anime to receive updates about new episodes via mail. Admins have the option to schedule emails.
  • The backend is built using Django REST, Redis, and Celery, and the frontend uses ReactJS and Redux
  • Since Django is an object-oriented framework and the database is PostgreSQL, OOPS and DBMS concepts were practically employed.
